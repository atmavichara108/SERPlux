# SERPlux — Инструкция для нового клиента

**Язык:** русский  
**Для кого:** заказчик (проектный менеджер, маркетолог)  
**Что это:** пошаговая инструкция по добавлению нового клиента в SERPlux

---

## Что вам нужно подготовить

1. **Аккаунт Topvisor** с доступом к своему проекту
2. **Google Drive** — где вы храните таблицы
3. **Данные от владельца сервера:**
   - URL сервера SERPlux (например, `https://serplux.mycompany.com`)
   - WEBHOOK_SECRET (опция безопасности)

---

## Пошаговая инструкция

### Шаг 1. Получите project_id из Topvisor

1. Откройте [Topvisor](https://topvisor.com)
2. Перейдите в **ваш проект**
3. В левом меню найдите **Snapshots API** (или **Снимки**)
4. Скопируйте **Project ID** (большое число, например `123456789`)
5. Сохраните его — пригодится на Шаге 5

### Шаг 2. Создайте копию мастер-таблицы

1. Откройте Google Drive
2. Найдите **мастер-таблицу SERPlux** (дал вам администратор)
3. **Правый клик** → **Создать копию**
4. Переименуйте копию в удобное вам имя (например, `SERPlux - Клиент ABC`)
5. Откройте новую таблицу — готова к работе

### Шаг 3. Установите скрипт Google Apps Script

1. В таблице нажмите **Расширения** → **Apps Script**
2. Удалите весь код, который там есть (выделите всё, Delete)
3. Вставьте свежий код из файла `apps_script.gs` (попросите у администратора или возьмите из репозитория)
4. Нажмите **Ctrl+S** (сохраните)
5. Закройте редактор Apps Script и вернитесь в таблицу

### Шаг 4. Инициализируйте шаблон листов

1. Вернитесь в редактор Apps Script (Расширения → Apps Script)
2. В левой панели выберите функцию **`initTemplateSheets`** из выпадающего меню
3. Нажмите **Run** (зелёная кнопка)
4. Разрешите доступ, когда попросит Google (это безопасно)
5. Подождите, пока не появится сообщение "Готово"
6. Закройте редактор — в таблице появятся листы:
   - ✓ Настройки
   - ✓ Данные (кэш выдачи)
   - ✓ Отчёт (матрица по гео)
   - ✓ Эталон разметки (кэш тональности)
   - ✓ Спорные (для будущих улучшений)
   - ✓ Лог (запись событий)

### Шаг 5. Установите URL сервера и секрет

1. На листе **Настройки** найдите первую таблицу
2. Нажмите **SERPlux** → **⚙ Настройки** → **Установить URL сервера**
3. Введите URL из пункта "Что вам нужно подготовить" (например, `https://serplux.mycompany.com`)
4. Нажмите **OK**
5. Затем **SERPlux** → **⚙ Настройки** → **Установить секрет**
6. Введите WEBHOOK_SECRET из пункта "Что вам нужно подготовить"
7. Нажмите **OK**

### Шаг 6. Добавьте клиента в систему

1. Нажмите **SERPlux** → **[К] Клиенты** → **Добавить клиента...**
2. Будет предложено ввести:
   - **client_id:** уникальный идентификатор (например, `client_abc`, только буквы, цифры, подчёркивание)
   - **Имя клиента:** для читаемости (например, `ABC Company`)
   - **Project ID Topvisor:** то число, которое вы скопировали на Шаге 1
   - **Поисковики:** выберите Google и/или Яндекс (✓ Google, ✓ Яндекс.ру)
   - **Гео:** выберите из списка Topvisor (например, Литва, Германия, Великобритания — можно несколько)
3. Нажмите **Сохранить**

### Шаг 7. Добавьте запросы (субъектов)

1. На листе **Настройки** в поле **client_id** выберите только что созданного клиента
2. Листы автоматически перезагрузятся
3. В листе **Данные** или через **SERPlux** → **[К] Клиенты** → **Управление запросами...**
4. Добавьте **запросы** (это имена субъектов, которых вы отслеживаете):
   - Одна строка = один запрос (например, `John Doe`, `ACME Inc`, `CEO Smith`)
   - Система автоматически будет собирать позиции этих запросов в поисковой выдаче
5. Нажмите **Сохранить**

### Шаг 8. Запустите первый сбор

1. Нажмите **SERPlux** → **▶ Запустить сбор**
2. Диалог покажет:
   - Клиент: выбранный client_id
   - Глубина: 10 (по умолчанию — количество позиций, которые собирать)
   - Разметка: включена (автоматическая)
   - Дата: сегодня
3. Нажмите **Да** — запуск начался
4. Статус: **SERPlux** → **⟳ Проверить статус** — в меню появится статус:
   - `running` — выполняется (может быть несколько минут)
   - `ok` — готово!
   - `error` — ошибка (свяжитесь с администратором)

### Шаг 9. Посмотрите результат

Когда статус становится `ok`, в таблице автоматически заполняются:

- **Лист «Данные»** — плоский формат с колонками:
  ```
  Дата | Поисковая система | Субъект/Запрос | Гео | Позиция | URL | Домен | Сниппет | Метка
  2026-07-12 | Google | John Doe | Литва | 1 | https://example.com/... | example.com | Текст сниппета | positive
  ...
  ```

- **Лист «Отчёт»** — матрица позиций по гео (цветная):
  ```
  Субъект        | Литва | Германия | ...
  John Doe (pos) | 1     | 5        | ...
  ACME Inc (pos) | 12    | 3        | ...
  ```
  - 🟢 Зелёный номер = positive (позитивная новость)
  - 🔴 Красный номер = negative (негативная новость)
  - 🟡 Жёлтый номер = neutral (нейтральная новость)

- **Лист «Эталон разметки»** — кэш тональности (пополняется автоматически)

- **Лист «Лог»** — события (когда и что произошло)

## Что дальше?

### Повторные запуски

1. **SERPlux** → **▶ Запустить сбор** (каждый день / по графику)
2. Результаты накапливаются в листе **Отчёт** (старые версии сдвигаются вниз, максимум 10 версий)
3. Лист **Данные** перезаписывается каждый раз новыми данными

### Изменение параметров

На листе **Настройки** вы можете менять:
- **depth:** 10, 20, 50 или 100 (сколько позиций собирать)
- **with_labels:** true/false (включить/выключить разметку тональности)
- **label_mode:** auto (кэш + сниппет, быстро) или deep (полная страница, дороже, будущее)
- **date:** today (сегодня) или конкретная дата YYYY-MM-DD

### Управление клиентами

- **Обновить список клиентов:** SERPlux → ⚙ Настройки → [>] Обновить список клиентов
- **Посмотреть гео Topvisor:** SERPlux → [К] Клиенты → Обновить гео из Topvisor
- **Добавить новый запрос:** на листе Данные вручную или через меню

## Если что-то не сработало

### Ошибка "Authentication failed"
- Проверьте **WEBHOOK_SECRET** (SERPlux → ⚙ Настройки → Установить секрет)
- Проверьте **URL сервера** (SERPlux → ⚙ Настройки → Установить URL сервера)

### Статус "running" долгое время
- Topvisor может быть медленным (особенно на больших глубинах)
- Проверьте лог: **SERPlux** → **⟳ Проверить статус** → посмотрите на время обновления

### Нет данных в листе "Данные"
- Проверьте, что client_id выбран на листе **Настройки**
- Проверьте, что запросы (субъекты) добавлены в профиль клиента
- Проверьте, что гео выбрано в профиле клиента

### Ошибка "Sheet not found"
- Убедитесь, что вы запустили **initTemplateSheets** (Шаг 4)
- Если листы потеряны, запустите initTemplateSheets ещё раз
