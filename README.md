
# SERPlux

Автоматический сбор поисковой выдачи Google и Яндекс по запросам и гео
через topvisor Snapshots API, разметка URL по тональности
(позитив / негатив / нейтрал) и выгрузка в Google Sheets с версионированием.

## Что делает
- Собирает ТОП выдачи (10/20/50/100) по связкам: запрос × гео × поисковая система
- Источник — topvisor (Google, Яндекс.ру, Яндекс.com)
- Размечает URL по тональности (списки + опционально LLM)
- Пишет результат в Google Sheets, накапливая историю по дате
- Запуск в один клик из самой таблицы

## Интерфейс
Управление и результат — внутри Google Sheets:
- лист "Настройки": запросы, гео, поисковая система, глубина, режим разметки
- кнопка "Запустить сбор"
- лист "Результаты": Дата | ПС | Запрос | Гео | Позиция | URL | Домен | Метка
- листы "Позитивные" / "Негативные"

## Архитектура
Модульная: config -> collector (topvisor) -> storage (кэш) ->
labeler (разметка) -> exporter (Sheets). См. AGENTS.md.

## Настройка
- Запросы / гео / ПС / глубина — лист "Настройки" в таблице
- Ключи — файл .env (см. .env.example)
- Подключение Google Sheets — service account, см. раздел ниже

## Источник данных и его ограничения
topvisor Snapshots: сбор ТОПа выдачи. Глубина зависит от настроек проекта
(прокидывается параметром). Сбор асинхронный: запуск проверки -> поллинг ->
получение снимка. Лимиты — по тарифу аккаунта topvisor.

## Разметка по тональности (LLM)
Простой режим — Gemini Flash, дешёвый (центы за прогон 450 URL).
Кэш URL: повторно встреченные ссылки не гоняются через LLM.
Сравнение провайдеров и расширенный режим — см. docs/llm-analysis.md.
